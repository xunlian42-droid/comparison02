操作説明：<br>
・トップページからcomparisonページに入る。<br>
・新規登録でユーザー登録する。<br>
・上の検索欄は作品タイトルのAND検索、下の検索欄は作品タイトルについているタグのOR検索。<br>
・除外語はタイトル検索の場合、タイトルの一部を入力し、該当タイトルを除くときに使う。<br>
(例:検索語に鬼滅の刃、除外語に舞台を入力し検索すると、テレビアニメ、劇場版、スペシャルイベントの鬼滅の刃がヒットする)<br>
・タグ検索の場合、除外したい該当タイトルについているタグの全体、もしくは一部を入れる。<br>
(例:検索語にサンライズ、除外語に20(2000年などのタグの一部)を入力し検索すると、90年代以前のサンライズ作品がヒットする)<br>
・ただし、タグはスクレイピングしたサイトの作品タグに依存する。<br>
・タグ一覧を表示ボタンは、スクレイピングしたタグの情報がリスト表示される。(どのタグがあるか確認はできるが検索には不向き)<br>
・タイトル検索、タグ検索、両方とも絞り込んだタイトルのリンクをクリックすると、該当ページの上部に遷移し、タイトルがハイライトされる。<br>
・検索、または比較ページに直接飛ぶと、五十音別で作品タイトルがリスト表示される。<br>
・作品タイトル上のリンクをクリックすると、ポップアップが開きその作品のタグ一覧がリスト表示される。<br>
・タグをクリックすると、そのタグを持つ比較表ページ内(ア行の作品比較)の作品タイトルが表示される。<br>
・タイトルリンクをクリックすると、そのタイトルにスクロールされてハイライトが光る。<br>
・dアニメストアとU-NEXTの欄は、◎をクリックするとサイトの作品ページに直接繋がる。<br>
・ポップアップ内に既存のタグとは別に、新規でタグを追加する。<br>
・追加タグは追加初期、未設定:○○になっているが、マイページで未設定欄(Role:)ポップアップ内で追加したタグ語(Name:)を編集することができる。<br>
・「この作品をタグから外す」ボタンで追加先のタイトルをタグから外すことができる。(外したらその場で戻すことはできない)<br>
・保存ボタンとキャンセルボタンは、(Role:)と(Name:)だけに対して行われる処理。<br>
